# Dynamic App Builder

Config-driven mini app generator.

## Features
- Dynamic UI generation
- Dynamic CRUD APIs
- PostgreSQL storage
- Authentication
- User-scoped data
- Multi-language
- CSV import
- Notifications
- Validation layer
- Error handling

## Tech
- Next.js
- TypeScript
- Prisma
- PostgreSQL
- NextAuth
- TailwindCSS

## Architecture
JSON Config → Validator → UI Renderer → API → DB
